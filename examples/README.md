# Example Files

This folder contains compact reviewer-facing examples and metadata templates.

## Included files

- `formulation_rheology_summary.csv`: formulation-level rheology summary table.
- `formulation_rheology_summary.xlsx`: Excel version of the same table.
- `foundation_model_variants.csv`: pretrained foundation encoders evaluated in the study.
- `expected_metadata_columns.csv`: expected input columns for running the benchmark scripts.
- `example_metadata_small.csv`: minimal example metadata for the three included CLSM images.
- `confocal_images/`: representative compressed CLSM PNG examples.

The included PNG images are for visual reference and repository demonstration. Full quantitative reproduction should use the original microscopy files and full metadata table.
